# Python Projects Collection

This repository contains a set of small Python practice projects, games, and utilities. Most of them are standalone scripts or small folders focused on one topic such as terminal games, file handling, APIs, or automation.

## Projects

### Folders

- `aim_trainer`  
  A reaction and mouse-accuracy training game.

- `alarm_app`  
  A simple alarm clock project that plays a sound at the chosen time.

- `backup.py`  
  A small backup utility that copies files into a backup folder.

- `basketball_nba_api`  
  A script that fetches or displays NBA-related data from an API.

- `exchange_currency`  
  A currency exchange or conversion tool.

- `mastermind_game`  
  A Python version of the Mastermind guessing game.

- `password_generator.py`  
  A password generator project for creating random passwords.

- `python_scripting`  
  A scripting exercise that finds game folders, copies them, creates metadata, and compiles Go game files when Go is available.

- `shortes_path_finder`  
  A shortest-path project that explores pathfinding logic.

- `slot_machine`  
  A slot machine game implemented in Python.

- `speed_typing_test`  
  A typing-speed test that measures how quickly text can be typed.

- `turtle_race`  
  A small race game built with Python's `turtle` module.

- `youtube_download`  
  A YouTube downloader utility.

### Standalone Scripts

- `choose_your_own_adventure.py`  
  A text-based branching story game.

- `dice_game_roll_more.py`  
  A dice game based on repeated rolling decisions.

- `math_test_with_timer.py`  
  A timed math quiz.

- `number_guesser.py`  
  A number guessing game.

- `password_manager.py`  
  A simple password manager that stores and retrieves passwords.

- `quiz_game.py`  
  A quiz game with predefined questions and answers.

- `replace_words_textfile.py`  
  A text-processing script that replaces words in a file.

- `rock_paper_scissors.py`  
  A classic rock-paper-scissors game.

## Notes

- Some projects use additional files such as `story.txt`, `text.txt`, `passwords.txt`, `key.key`, audio files, or downloaded media.
- Several folder names include `.py`, but they are directories rather than single Python files.
- This repository is mainly a learning and practice collection rather than one integrated application.
